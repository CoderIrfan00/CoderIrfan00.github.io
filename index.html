Featured Projects

CertiResumeGenerator

I developed a certificate management website using the MERN stack that allows users to upload and manage their certificates. The platform supports both PDF and image uploads and uses Optical Character Recognition (OCR) to extract text from uploaded certificates for easy access and resume creation. The site features user authentication, a secure file upload system, and a dashboard where users can view, organize, and download their certificates. It includes error handling, PDF-to-image conversion, and responsive design for a seamless user experience.

JavaCoder - Professional Educational Platform

JavaCoder is a comprehensive educational platform built using Next.js, designed to provide high-quality courses for both school-level students (grades 8-12) and undergraduate students. For school students, JavaCoder offers in-depth lessons in core subjects such as Mathematics, English, Science, and Social Science. For undergraduate students, the platform delivers cutting-edge technical courses, including HTML, CSS, JavaScript, React.js, and Next.js, with a focus on project-based learning. The platform is designed with a clean and modern interface, providing features like personalized dashboards, dynamic course pages, and interactive learning paths. With future updates planned for expanding its offerings, JavaCoder aims to be a go-to destination for students looking to enhance their academic and technical skills.
